# Battle Bee Game Tracker

**Commander Format Life Tracker for iOS**

Battle Bee Game Tracker is a life tracking app for Commander format card games — built for the table, designed for 2, 4, or 6 players.

---

## Features

- **Life tracking** — Tap left/right to adjust life totals; hold for a quick +/- menu
- **Commander damage** — Track damage received from each opponent; auto-eliminates at 21
- **Poison counters** — Tracks infect; auto-eliminates at 10
- **Multi-player layouts** — 2, 4, and 6 player modes with rotated panels so everyone faces their own
- **Finger chooser** — All players place a finger on screen; one is randomly chosen to go first
- **Custom starting life** — 20, 30, 40, 60, or any custom value
- **Themes** — 6 colour themes: Arcane, Ocean, Inferno, Nature, Gold, Void
- **GIF backgrounds** — Pick a GIPHY GIF for each player's panel
- **Player profiles** — Custom name and emoji per player
- **Game history & stats** — Log games with deck names, commanders, and winners; view win rates and leaderboard
- **No account, no ads, no internet required** — all data stays on device

## Requirements

- iOS 16.0+
- Xcode 15+
- Swift 5.9+

---

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/BattleBeeGames/BattleBeeGameTracker.git
   ```
2. Open `Battle Bee Game Tracker.xcodeproj` in Xcode
3. Select your target device or simulator
4. Build and run (`⌘R`)

No external dependencies or package manager setup required.
